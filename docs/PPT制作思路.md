## 二、逐页内容设计

### 第1页：封面
- 标题：无人机集群配送路径规划
- 副标题：基于贪心、遗传、模拟退火的对比优化
- 组员姓名、学号
- 指导老师
- 日期

### 第2页：目录
- 简洁4-5行即可
- 项目背景 → 数据描述 → 算法设计 → 实验结果 → 总结

### 第3页：项目背景与问题定义

**内容**：
- 1句话背景：低空物流是AI优化的典型应用
- 问题定义图：配送中心(50,50) + 分散客户点 → 最小化总飞行距离
- 约束条件卡片（3个图标式呈现）：
  - 载重 ≤ 20kg
  - 单趟里程 ≤ 200单位
  - 所有客户必须配送

**图表**：客户点分布散点图（标注配送中心为红星）

### 第4页：数据描述与预处理

**内容**：
- 左侧：数据参数表（客户数20、坐标范围、重量范围、随机种子42）
- 右侧：客户点分布散点图（按重量着色）
- 底部：清洗流程（3步箭头图：边界校验 → 异常剔除 → 去重）

**图表**：按重量着色的客户分布散点图 + 重量分布直方图

### 第5页：算法总览

**内容**：三种算法对比框架图（表格形式）

| 算法 | 类型 | 核心策略 | 特点 |
|------|------|---------|------|
| 贪心 | 构造启发式 | 每次选最近可行客户 | 快速、基准线 |
| 遗传 | 元启发式 | 种群进化+自然选择 | 全局搜索、可并行 |
| 模拟退火 | 元启发式 | 温度递减+概率接受 | 跳出局部最优 |

**图表**：一个统一流程图 —— 编码(排列) → 解码(打包路线) → 评估(总距离+惩罚)

### 第6页：贪心算法（基线方法）

**内容**：
- 一句话原理
- 贪心路径图（展示最优一次运行结果）
- 关键指标：总距离 716.65，5趟

**图表**：贪心路径图 + 各趟次载重/里程柱状图

### 第7页：遗传算法 — 编码与解码

**内容**：
- 左侧：编码示意图（20个客户索引的排列 → 基因序列）
- 右侧：解码示意图（排列按序贪心打包 → 各趟次路线）
- 关键点：排列编码天然保证每个客户只访问一次

**图表**：编码/解码流程示意图（用箭头和颜色标注）

### 第8页：遗传算法 — 进化过程与结果

**内容**：
- 遗传算子卡片：锦标赛选择 + 有序交叉 + 洗牌变异 + 精英保留
- 超参数：pop=200, gen=200, cx=0.8, mut=0.3, elite=10
- 关键结果：最优距离 549.58，4趟，较贪心提升 23.3%

**图表**：进化曲线（最优适应度 + 平均适应度随代数下降）

### 第9页：模拟退火 — 核心机制

**内容**：
- 退火类比示意图（高温→低温 = 广搜→精调）
- 三种邻域算子：Swap / Reverse / Insert（图示）
- Metropolis准则公式（简洁呈现）
- 关键参数：T0=500, α=0.98, 3次重启

**图表**：邻域算子示意图 + 温度下降曲线

### 第10页：模拟退火 — 搜索过程与结果

**内容**：
- 3次重启的搜索轨迹（最优距离随迭代下降）
- 关键结果：最优距离 532.99，4趟，较贪心提升 25.6%

**图表**：3次重启的收敛对比图 + 总览路径图

### 第11页：实验结果 — 三算法全面对比（核心页）

**内容**：
- 核心对比表（加粗最优值）：

| 指标 | 贪心 | 遗传 | 模拟退火 |
|------|------|------|---------|
| 总飞行距离 | 716.65 | 549.58 | **532.99** |
| 总趟次 | 5 | 4 | **4** |
| 提升幅度 | — | 23.3% | **25.6%** |

- 总结语句：两种元启发式算法均显著优于贪心基线

**图表**：三算法路径并列对比图 + 总距离柱状图

### 第12页：可视化展示

- 遗传算法总览大图（暗色主题，展示路线叠加效果）
- 或选模拟退火路径图
- 可选：单架无人机详情（选最有代表性的）

### 第13页：总结与展望

**总结**：
- 完成了三种算法的实现与系统对比
- 模拟退火最优（532.99），遗传算法次之（549.58），均远超贪心基线
- 统一排列编码框架使算法可互换对比

**展望**：
- 引入时间窗约束（更贴近实际配送场景）
- 考虑续航衰减模型（载重影响能耗）
- 尝试强化学习方案（DQN/PPO求解VRP）
- 开发交互式可视化平台

### 第14页：致谢 / Q&A
- 感谢老师指导
- 欢迎提问

## 三、PPT制作规范

### 视觉风格
- 配色方案：建议统一使用暗色主题（与代码可视化一致），或简洁白底
- 字体：中文用微软雅黑或思源黑体，英文用 Arial
- 字号：标题 ≥ 28pt，正文 ≥ 18pt，图表标注 ≥ 12pt
- 避免大段文字：用要点列表（每条≤2行），关键数字加粗放大

### 图表复用
以下图表可直接从 `outputs/` 文件夹截图使用：
- `comparison_all_*.png` → 第11页三算法对比
- `drone_delivery_ga_*_overview.png` → 第12页可视化
- `drone_delivery_ga_*_evolution.png` → 第8页进化曲线
- `drone_delivery_sa_*_convergence.png` → 第10页收敛对比

需要额外生成的图表（建议用Python脚本快速生成）：
- 客户重量分布直方图 → 第4页
- 三算法总距离柱状对比图 → 第11页
- 算法运行时间对比图 → 第11页

### 动画建议（适度使用）
- 算法流程可做简单逐步出现动画
- 退火温度下降可用渐变动画
- 避免花哨过场，保持专业
